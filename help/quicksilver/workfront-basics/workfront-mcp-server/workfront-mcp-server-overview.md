# Visão geral do servidor Adobe Workfront MCP

O servidor MCP (Model Context Protocol) do [!DNL Adobe Workfront] permite trabalhar com os dados do Workfront por meio de conversação em linguagem natural em um assistente de IA compatível. Em vez de clicar no Workfront para encontrar um projeto, atualizar uma tarefa ou verificar uma aprovação, você solicita o assistente de IA em inglês simples, e o assistente de IA faz o trabalho para você.

Você não precisa saber a API do Workfront nem nada sobre o MCP para usar o servidor MCP do Workfront. Configure o assistente de IA para se comunicar com a Workfront uma vez e, em seguida, comece a perguntar.

## Configurar e usar o servidor MCP do Workfront

Antes de usar o servidor MCP do Workfront, um administrador do Workfront deve habilitar o acesso ao servidor MCP no Workfront e você deve conectar seu assistente de IA ao Workfront. Para obter mais informações, consulte [Configurar o servidor MCP do Adobe Workfront](/help/quicksilver/workfront-basics/workfront-mcp-server/configure-workfront-mcp-server.md).

Após a configuração, você pode solicitar que o assistente de IA localize, crie, atualize e gerencie itens do Workfront. Para obter mais informações sobre ações disponíveis, exemplos de solicitações e tratamento de dados, consulte [Usar o servidor MCP do Adobe Workfront](/help/quicksilver/workfront-basics/workfront-mcp-server/use-workfront-mcp-server.md).
