# Introdução a [!UICONTROL Início]

Há duas maneiras de visualizar seu trabalho na [!UICONTROL Início] da [!UICONTROL Lista de Trabalho] ou o Calendário.

## Lista de trabalhos

O [!UICONTROL Lista de Trabalho] O permite visualizar todo o trabalho atribuído em um local:

![[!UICONTROL Lista de Trabalho] painéis](assets/worklist-and-right-panel-home.png)

R. Use o painel esquerdo para filtrar, classificar e selecionar o trabalho atribuído a você.

B. Depois de selecionar um item de trabalho no painel esquerdo, use o painel direito para interagir com campos personalizados, fazer atualizações e tempo de log.

Para obter mais informações sobre como usar o [!UICONTROL Lista de Trabalho Doméstica], consulte os seguintes artigos:

* [Exibir itens no [!UICONTROL Lista de Trabalho] no [!UICONTROL Início] area](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md)
* [Criar itens de trabalho a partir do [!UICONTROL Início] area](../../../workfront-basics/using-home/using-the-home-area/create-work-items-in-home.md)

## Calendário

O Calendário permite visualizar seu trabalho ao exibir quando os itens de trabalho devem aparecer e permitir que você bloqueie o tempo para concluí-los:

![](assets/calendar-home-nwe-350x114.png)

A. Use o [!UICONTROL Calendário] exibir para criar um mapa visual do trabalho que deve ser concluído. Basta clicar e arrastar os itens de trabalho da [!UICONTROL Lista de Trabalho] no Calendário para bloquear o tempo de trabalho.

B. Use a opção de sincronização do Calendário para sincronizar com seu [!DNL Outlook] calendário. Gerencie reuniões e trabalhe em um só lugar.

C. Use o [!UICONTROL Detalhes] para abrir um novo painel direito, onde você pode exibir mais informações sobre o item de trabalho.

D. Use a barra de vencimento para acompanhar quando o trabalho atribuído a você está vencido.

Para obter mais informações sobre como usar o Calendário principal, consulte [Use o [!UICONTROL Calendário doméstico] exibir](../../../workfront-basics/using-home/using-the-home-area/use-home-calendar-view.md).

## Encontre seu trabalho

O [!UICONTROL Início] é o seu balcão único para localizar tarefas, emissões e aprovações atribuídas a você. As opções de filtro e classificação incorporadas ao [!UICONTROL Início] trabalhe em conjunto para ajudá-lo a trabalhar da maneira que quiser. Você pode usar o [!UICONTROL Classificar por] para agrupar itens e usar o Filtro para se concentrar no trabalho que precisa ser concluído.

>[!NOTE]
>
>As opções de filtro e classificação são armazenadas no navegador. Se você usar consistentemente o mesmo navegador no mesmo computador (e não limpar os dados do site), os filtros e a classificação não serão alterados, mas se você alternar navegadores ou computadores, os filtros e a classificação serão diferentes.

### Agrupar itens de trabalho semelhantes com &quot;[!UICONTROL Agrupar por]Opção &#39;

O [!UICONTROL Classificar por] permite agrupar itens semelhantes na Lista de Trabalho. Você pode classificar por:

* [!UICONTROL Conclusão planejada]
* [!UICONTROL Início planejado]
* [!UICONTROL Data de confirmação]
* [!UICONTROL Projeto]
* [!UICONTROL Minhas Prioridades]

Para obter mais informações sobre como usar agrupamentos, consulte [Exibir itens no [!UICONTROL Lista de Trabalho] no [!UICONTROL Início] area](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/group-by-drop-down-expanded-in-home-with-planned-start-date-nwe-350x273.png)

### Restrinja seu foco com filtros

O [!UICONTROL Lista de Trabalho] O filtro permite limitar o foco a itens de trabalho específicos.

A seguir estão exemplos de como você pode filtrar itens exibidos na Página inicial:

* Filtrar por tipo de item e estado.

   Por exemplo, se você deseja exibir todas as tarefas, é possível selecionar a variável [!UICONTROL Tarefas] filtro. Se desejar ser mais específico do que isso e exibir somente as tarefas que estão prontas para você começar a trabalhar, selecione a variável [!UICONTROL Pronto para começar] filtros sob o [!UICONTROL Tarefas] filtro.

* Filtrar somente por tipo de item.

   Por exemplo, você pode selecionar [!UICONTROL Problemas] para ver todos os problemas em todos os estados ([!UICONTROL Trabalhando em] ou [!UICONTROL Solicitado]), ou [!UICONTROL Aprovações] para ver todos os itens de trabalho, solicitação de acesso, folha de ponto, documento e aprovações de prova.

* Filtrar somente por status.

   Por exemplo, você pode selecionar a variável [!UICONTROL Concluído] para exibir tarefas e problemas que foram concluídos. Isso inclui tarefas Pessoais, mas não inclui aprovações.

Para obter mais informações sobre como usar filtros, consulte [Exibir itens no [!UICONTROL Lista de Trabalho] no [!UICONTROL Início] area](../../../workfront-basics/using-home/using-the-home-area/display-items-in-home-work-list.md).

![](assets/displaying-work-items-filters-nwe-350x401.png)

### Acessar trabalhos atribuídos à sua equipe

O [!UICONTROL Início] tem um agrupamento permanente dedicado às solicitações da equipe com um link direto para a página de solicitação da equipe. Use este agrupamento para exibir e acessar solicitações de qualquer uma das equipes nas quais você está.

>[!NOTE]
>
>O [!UICONTROL Filtro] e [!UICONTROL Classificar por] as opções não afetam o [!UICONTROL Solicitações de equipe] agrupamento. Esse agrupamento fica visível desde que você tenha trabalho atribuído à sua equipe.

Para obter mais informações sobre como acessar solicitações de equipe, consulte o artigo [[!UICONTROL Gerenciar] solicitações de equipe e de trabalho na [!UICONTROL Início] area](../../../workfront-basics/using-home/using-the-home-area/manage-work-and-team-requests-home.md).

![](assets/team-requests-expanded-home-group-by-drop-down-nwe-350x314.png)

### Monitorar o trabalho submetido

Acompanhe o trabalho enviado para aprovação diretamente do [!UICONTROL Lista de Trabalho]. Você pode lembrar o aprovador sobre o trabalho que precisa de aprovação. Você também pode cancelar a aprovação, se necessário.

>[!NOTE]
>
>O [!UICONTROL Filtro] e [!UICONTROL Classificar por] as opções não afetam o [!UICONTROL Aprovações que enviei] agrupamento. Esse agrupamento fica visível, desde que você tenha trabalho aguardando aprovação.



![](assets/approvals-expanded-home-group-by-drop-down-nwe-350x415.png)

## Priorize o que é importante para você

O [!UICONTROL Lista de Trabalho] permite que você apareça e priorize o trabalho que é importante para você com a variável [!UICONTROL Minha classificação de prioridade por] opção. Você pode adicionar até 20 itens à sua [!UICONTROL Minha prioridade] lista. Use o [!UICONTROL Mais] para adicionar itens de trabalho à lista de prioridades.

>[!NOTE]
>
>Qualquer item de trabalho adicionado a [!UICONTROL Minha prioridade] são visíveis apenas para você, e ninguém pode priorizar o trabalho para você.

Para obter mais informações sobre como usar [!UICONTROL Minha prioridade], consulte [Priorize o trabalho no [!UICONTROL Início] area](../../../workfront-basics/using-home/using-the-home-area/prioritize-work-in-home.md).

![](assets/prioritizing-in-home-group-by-drop-down-nwe-350x412.png)

## Faça isso

Use o painel direito em [!UICONTROL Início] para atualizar e concluir seu trabalho.

### Trabalhe em um espaço personalizado para você

Seu [!DNL Adobe Workfront] O administrador pode adicionar até 16 campos personalizados para facilitar a localização e a atualização das informações necessárias. A maioria dos campos é editável, portanto, não é necessário navegar até o item de trabalho.

Para atualizar qualquer um dos campos nessa área, basta clicar no campo .

>[!NOTE]
>
>Você não pode atualizar o [!UICONTROL Data de confirmação] nesta área.

![](assets/16-fields-home-350x137.png)

### Atualize os colegas de trabalho com informações importantes

Comunique-se rápida e eficientemente com os colegas de trabalho da [!UICONTROL Início] área. Faça perguntas aos colegas de trabalho, notifique-os de atualizações recentes ou responda a comentários diretos. Além disso, você pode atualizar outros sobre o progresso que você está fazendo ao:

* Atualização do [!UICONTROL Data de confirmação]

   <!--
  <note type="note">
  This is the only place you can update the Commit Date in Home.
  <br>
  </note>
  -->

* Relatar como você acha que as coisas estão acontecendo
* Atualização da barra de porcentagem concluída

Para obter mais informações sobre a comunicação no [!UICONTROL Início] área, consulte [Atualizar trabalho](../../../workfront-basics/updating-work-items-and-viewing-updates/update-work.md).

![](assets/updates-home-350x198.png)

### Registre seu tempo

Acompanhe as horas que você trabalhou em itens de trabalho individuais. Você pode escolher um tipo de hora geral, como [!UICONTROL Hora da Tarefa]ou você pode escolher um tipo de hora específico do projeto configurado pelo seu [!DNL Workfront] administrador.

Para obter mais informações sobre como registrar o tempo no [!UICONTROL Início] , consulte a seção [Início](../../../timesheets/create-and-manage-timesheets/log-time.md#home) no artigo [Tempo de registro](../../../timesheets/create-and-manage-timesheets/log-time.md).

![](assets/log-time-home-350x181.png)
