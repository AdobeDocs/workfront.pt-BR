# Gerencie sua instância do Adobe Workfront Planning como um produto independente

>[!IMPORTANT]
>
>As informações neste artigo se referem ao Adobe Workfront Planning, quando adquirido como um produto independente. Consulte este artigo quando sua empresa comprou um pacote somente do Adobe Workfront Planning e não comprou um pacote do Workfront Workflow.
>
>Para obter informações sobre o Adobe Workfront Planning quando adquirido junto com um pacote do Workfront, consulte [Introdução ao Adobe Workfront Planning](/help/quicksilver/planning/general/planning-overview.md).

Como Administrador do Workfront Planning, você pode gerenciar sua instância do Adobe Workfront Planning.

É possível definir as seguintes configurações para o Workfront Planning:

* Trimestres personalizados
* Informação de Cliente
* Preferências
* Equipes

  Para obter informações, consulte [Gerenciar equipes no Adobe Workfront Planning como um produto independente](/help/quicksilver/planning/planning-sta/manage-teams-in-planning-sta.md).
* Usuários

  Para obter informações, consulte [Gerenciar usuários no Adobe Workfront Planning como um produto independente](/help/quicksilver/planning/planning-sta/manage-users-in-planning-sta.md).

A configuração do Workfront Planning como um produto independente é semelhante à definição de algumas das configurações no Adobe Workfront. No entanto, há limitações para quais informações estão disponíveis no Workfront Planning.

## Fazer logon como outro usuário

Como Administrador do Planning, você pode fazer logon como outro usuário para solucionar problemas para ele. Fazer logon como outro usuário no Workfront Planning é semelhante a fazer logon como outro usuário no Workfront.

Para obter informações, consulte [Fazer logon como outro usuário](/help/quicksilver/administration-and-setup/add-users/create-and-manage-users/log-in-as-another-user.md).

## Configurar trimestres personalizados

Talvez você queira criar trimestres personalizados se os trimestres de sua organização forem baseados em critérios específicos diferentes das datas do calendário (como dias úteis ou dias de compras).

Você pode configurar até 100 trimestres para sua instância do Workfront Planning. Você pode usar esses trimestres personalizados na exibição de linha do tempo.

A configuração de trimestres personalizados no Workfront Planning é semelhante à configuração de trimestres personalizados no Workfront.

Para obter informações, consulte [Habilitar trimestres personalizados](/help/quicksilver/administration-and-setup/set-up-workfront/configure-system-defaults/enable-custom-quarters-projects.md).

## Configurar informações do cliente

Como parte da configuração do produto Adobe Workfront Planning, você pode gerenciar detalhes sobre sua organização na seção Informações do cliente da Configuração.

Configurar a seção Informações do Cliente no Workfront Planning é semelhante à configuração no Workfront.

Para obter informações, consulte [Configurar informações básicas do sistema](/help/quicksilver/administration-and-setup/get-started-wf-administration/configure-basic-info.md).


## Configurar as preferências do sistema

Como Administrador do Planning, você pode configurar preferências para seu sistema Workfront Planning.

As alterações feitas na área Preferências do sistema da Configuração afetam todos os usuários no sistema e sua experiência no Workfront Planning.

Recomendamos que você configure suas preferências de sistema durante o Workfront Planning como uma implementação de produto independente e só ocasionalmente as revise depois disso.

A configuração das preferências do sistema do Workfront Planning é semelhante à configuração delas no Workfront.

Para obter informações, consulte [Configurar preferências do sistema](/help/quicksilver/administration-and-setup/manage-workfront/security/configure-security-preferences.md).
