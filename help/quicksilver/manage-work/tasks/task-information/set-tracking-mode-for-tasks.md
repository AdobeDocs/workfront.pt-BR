## Definir modo de rastreamento para tarefas

Para definir o modo de rastreamento:

1. Vá para a tarefa para a qual deseja definir o modo de rastreamento.
1. Clique no botão **Mais** ícone ![](assets/qs-more-icon-on-an-object.png)ao lado do nome da tarefa, em seguida, clique em **Editar**.

   A caixa de diálogo Editar tarefa é aberta.

1. No **Configurações** use a seção **Modo de rastreamento** menu suspenso para selecionar o Modo de rastreamento da tarefa.

   ![](assets/tracking-mode-field-in-edit-task-settings-nwe-350x233.png)

1. Selecione dentre as seguintes opções:

   * O usuário deve atualizar (esta é a opção padrão)
   * Assume No Horário
   * Ignorar avisos de atrasos
   * Completar automaticamente
   * Predecessor

   Para obter mais informações sobre as opções do modo de rastreamento, consulte [Visão geral do modo de rastreamento de tarefas](../../../manage-work/tasks/task-information/task-tracking-mode.md)

1. Clique em **Salvar**.
