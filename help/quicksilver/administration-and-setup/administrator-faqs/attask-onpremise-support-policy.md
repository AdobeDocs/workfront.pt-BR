# Política de suporte AtTask OnPremise

A Adobe Workfront migrou para um modelo 100% de software como serviço e interrompeu as vendas de software no local em 31 de dezembro de 2011. A partir de 2014, o AtTask OnPremise não é mais suportado, com exceção de problemas relacionados à chave de licença. O aplicativo local não está mais disponível para download ou instalação.

Se você já tiver o aplicativo e precisar reinstalar o OnPremise, baixe o guia de instalação .

## Atualizar a chave de licença no local

Se precisar de uma nova chave de licença, ligue para o Suporte ao cliente da Workfront pelo telefone 844-306-HELP(4357).

Depois de obter uma nova chave de licença,

1. Interrompa o servidor atTask.
1. Renomeie o arquivo license.key atual (localizado na pasta AtTaskDoc).
1. Copie o novo arquivo license.key para o local.
1. Reinicie o atTask Server.
