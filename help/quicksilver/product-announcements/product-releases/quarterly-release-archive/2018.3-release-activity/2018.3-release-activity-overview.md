### A API versão 10 já está disponível na Visualização {#api-version-10-now-available-in-preview}

A API versão 10 já está disponível no ambiente de Visualização. Ele será disponibilizado no ambiente de Produção com a versão 18.3. Para obter mais informações, consulte [Novidades na API versão 10](../../../../wf-api/api/new-api-version-10.md). 

### A versão 3 da API não é mais suportada {#api-version-3-no-longer-supported}

O suporte para a API versão 3 está terminando em novembro de 2018. Para obter mais informações sobre o cronograma de suporte para versões da API do Workfront, consulte [Controle de versão da API e cronograma de suporte](../../../../wf-api/api/api-version-support-schedule.md).

### O suporte para TLS está terminando {#tls-support-is-ending}

Para fornecer segurança ideal, a Workfront exige que todas as integrações e tráfego da Web que usam TLS 1.0 ou anterior sejam atualizadas para usar TLS 1.1 ou posterior. Essa alteração já está em vigor no ambiente de Visualização.

Para obter mais informações, consulte  [TLS 1.2 a ser necessário no Adobe Workfront](../../../../product-announcements/announcements/announcement-archive/tls-1-disabled.md).

### O suporte a JSONP está terminando {#jsonp-support-is-ending}

Como JSONP (Javascript com preenchimento) é um padrão antigo com vulnerabilidades de segurança conhecidas, a Workfront não será mais compatível com JSONP a partir de novembro de 2018. Para obter mais informações, consulte [Encerramento do suporte para JSONP](../../../../wf-api/api/ending-support-jsonp.md).

### Remoção do aplicativo Flash {#flash-application-removal}

Toda funcionalidade que depende da tecnologia de Flash está planejada para ser removida da Workfront com a versão 2018.3 e substituída por uma nova solução.

Para obter informações detalhadas sobre as ferramentas de Flash herdadas e suas substituições, consulte [Substituição de ferramentas baseadas em Flashes no Adobe Workfront](../../../../product-announcements/announcements/announcement-archive/replace-flash-tools.md).

### Várias opções de logon único a serem removidas {#various-single-sign-on-options-to-be-removed}

As seguintes opções de Logon único (SSO) agora são suportadas apenas em uma capacidade limitada; serão retirados do produto no segundo semestre de 2019:

* LDAP

   Para obter mais informações sobre como usar o LDAP com o Workfront, consulte [Configurar o Adobe Workfront com o LDAP](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* Active Directory

   Para obter mais informações sobre o uso do Ative Diretory com o Workfront, consulte [Configurar o Adobe Workfront com o Ative Diretory](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

* SAML 1.1

   Para obter mais informações sobre o uso do SAML 1.1 com Workfront, consulte [Configurar o Adobe Workfront com o SAML 1.1](../../../../administration-and-setup/manage-workfront/security/removal-various-sso-options.md).

A seguir, a linha do tempo de remoção para essas opções de SSO:

* **Suporte limitado:** Agosto de 2018 - Janeiro de 2019

   As principais questões serão corrigidas; problemas de baixa gravidade não serão resolvidos.

* **Substituição:** Janeiro de 2019 - agosto de 2019

   Não suportado.

* **Não está mais disponível no Workfront:** Agosto de 2019

Essas alterações estão sendo feitas em preparação para a experiência de autenticação aprimorada, que atualmente está disponível somente para um número limitado de clientes.

### Atualização herdada do Workfront Fusion - 20 de setembro de 2018 {#legacy-workfront-fusionupdate-september-20-2018}

O Workfront Fusion herdado é uma plataforma de integração que permite a conexão do Workfront a vários aplicativos de terceiros.

Com a atualização mais recente, o Legacy Workfront Fusion agora oferece um design aprimorado que torna o processo de criação de novos FLOs mais intuitivo. As mudanças específicas incluem:

* **Novo FLO:** Clicar no novo FLO agora leva os usuários para a tela Designers , em vez de primeiro solicitar a escolha do conector. Essa alteração ajuda você a se orientar para a FLO geral e entender que qualquer FLO deve começar com um evento.
* **Adicionar função:** O seletor de função foi atualizado para facilitar a localização e a seleção da função desejada. As descrições das funções, juntamente com campos de entrada e saída, foram atualizadas para adicionar orientação intuitiva. A nova seção &quot;Mais popular&quot; ajuda você a começar a usar as funções mais usadas.
