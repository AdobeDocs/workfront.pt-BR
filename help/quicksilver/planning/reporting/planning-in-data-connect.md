# Conexão de Dados do Workfront para Workfront Planning

O Workfront Data Connect fornece acesso aos dados do Workfront, incluindo do Adobe Workfront Planning, usando a tecnologia Snowflake, permitindo que você responda a perguntas críticas para os negócios e meça o valor de seus processos. Os clientes corporativos podem analisar seus dados usando ferramentas de visualização de dados diretamente de um data lake da Workfront ou exportar dados do Workfront para um data warehouse de propriedade do cliente.

Os assinantes poderão conectar sua ferramenta de inteligência de negócios escolhida (Tableau, Power BI, Domo, Looker etc.) ao Data Connect para criar visualizações de dados com os seguintes tipos de dados:

* Dados de estado atual de baixa latência
* Alterar registros de eventos para comparações point-in-time
* Instantâneos diários de histórico para análise de tendência

Para obter informações sobre como criar uma conta de leitor com o Snowflake e estabelecer uma conexão com o Workfront Data Connect, consulte os seguintes artigos:

* [Criar uma conta ou conexão de leitor para o Snowflake](/help/quicksilver/reports-and-dashboards/data-lake/create-a-reader-account.md)
* [Estabelecer uma conexão com a conexão de dados do Workfront](/help/quicksilver/reports-and-dashboards/data-lake/share-data-externally.md)


Para obter informações sobre o Planning no dicionário de dados do Workfront Data Connect, consulte o [dicionário de dados do Workfront Data Connect](/help/quicksilver/reports-and-dashboards/data-lake/data-dictionary.md).
