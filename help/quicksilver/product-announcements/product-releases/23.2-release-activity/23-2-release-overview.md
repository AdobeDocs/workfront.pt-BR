# Visão geral da versão 23.2

Esta página fornece informações sobre a funcionalidade incluída na versão 23.2 programada para a semana de 3 de abril de 2023.

As melhorias nesta página estão disponíveis no ambiente de Visualização. Esta página será atualizada com melhorias adicionais, à medida que a versão 23.2 se aproxima da versão de produção planejada.

Para obter dicas sobre como se preparar para a próxima versão, consulte [Preparar para uma versão trimestral do Adobe Workfront](/help/quicksilver/product-announcements/product-releases/release-readiness.md).

## Webinário de versão

Em vez de um webinário de versão, convidamos você para se registrar no Adobe Summit para assistir às seguintes sessões do Workfront:

[Experiências impactantes: Da criação de conteúdo à personalização](https://reg.adobe.com/flow/adobe/as23/sessions/page/catalog/session/1661982243928001D5Z9) em 21 de março de 2023, para aprender novas maneiras de criar, gerenciar e personalizar conteúdo e fornecer experiências impactantes com mais eficiência. Esta sessão será transmitida ao vivo para públicos-alvo virtuais.

[O futuro do Adobe Workfront](https://reg.adobe.com/flow/adobe/as23/sessions/page/catalog/session/1661982239896001DoHU) em 23 de março de 2023, para ver a visão do Workfront e seu lugar no ecossistema do Adobe e dar uma espiada no futuro roteiro. Essa sessão será registrada e disponibilizada posteriormente como conteúdo sob demanda.

<span class="preview">Os recursos fora do ciclo (aqueles lançados para Produção antes da data de lançamento 23.2) são destacados em amarelo.</span>

Para obter mais informações, consulte o [Página Atualizações da versão de treinamento](https://one.workfront.com/s/training-release-updates).

### Funcionalidade que será removida em breve do Workfront

A seguinte funcionalidade será removida em breve do Workfront:

#### **As páginas externas do painel não são mais compatíveis com URLs do painel**

Para melhorar a estabilidade do painel e os tempos de carregamento, as páginas externas em painéis não poderão mais fazer referência às URLs do painel. Os administradores de sistema para contas que contêm qualquer página externa que faça referência a painéis receberão uma notificação no aplicativo para alertá-los sobre essa alteração. Para obter mais informações, consulte [Incorporar uma página externa em um painel](/help/quicksilver/reports-and-dashboards/dashboards/creating-and-managing-dashboards/embed-external-web-page-dashboard.md).

#### **Alterações na API padrão do Workfront**

Para garantir que as chamadas de API possam aproveitar a funcionalidade mais atualizada da API do Workfront, estamos atualizando a API padrão. Quando uma chamada de API não especifica uma versão da API, a chamada é feita para a API padrão.

Agora, a API padrão reflete a versão mais recente da API. A partir de agora, atualizaremos a API padrão sempre que uma nova versão da API for lançada, para que as chamadas para a API padrão sempre usem a versão mais recente da API.

Anteriormente, a API padrão usava a API do Workfront versão 2.0, que foi substituída.

Se sua organização estiver usando atualmente a API padrão obsoleta, o administrador do Workfront recebeu uma mensagem da Central de anúncios com mais instruções relacionadas à API padrão.

Para obter mais informações, consulte [Atualizar integrações que usam o controle de versão padrão da API](/help/quicksilver/wf-api/api/update-default-api-versioning.md).
