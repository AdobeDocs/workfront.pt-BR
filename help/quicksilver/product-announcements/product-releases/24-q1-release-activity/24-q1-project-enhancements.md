# Aprimoramentos do projeto do primeiro trimestre de 2024

Esta página descreve todos os aprimoramentos de projeto feitos com a versão do primeiro trimestre de 2024 para o ambiente de Visualização. Esses aprimoramentos serão disponibilizados no ambiente de produção na versão do primeiro trimestre de 2024.

Para obter uma lista de todas as alterações disponíveis neste momento no ciclo de lançamento do primeiro trimestre de 2024, consulte [Visão geral de lançamentos do primeiro trimestre de 2024](/help/quicksilver/product-announcements/product-releases/24-q1-release-activity/24-q1-release-overview.md).

## Atribuições inteligentes mais relevantes

>[!NOTE]
>
>Versão prévia: 21 de dezembro de 2023; Produção para versão rápida: com a versão 24.3 (março de 2024).
>
>_Este recurso foi removido da Visualização e da Produção de versão rápida._

Alteramos o algoritmo que o Workfront usa para calcular e sugerir atribuições inteligentes para tarefas. O novo algoritmo se aplica às seguintes áreas no Workfront em que você atribui uma tarefa: listas de tarefas, a área Atribuições no cabeçalho da tarefa, Início e o painel Resumo.

Agora, o algoritmo examina as seguintes informações quando sugere atribuições inteligentes:

* Semelhanças entre nomes de tarefa, projeto e portfólio.

* Frequência na qual uma atribuição é feita a uma tarefa com um nome semelhante.

Se o novo algoritmo não encontrar correspondências, os critérios existentes para calcular as atribuições inteligentes serão aplicados.

Considere também o seguinte:

* O nome da lista de atribuições inteligentes em uma lista de tarefas foi alterado na lista de tarefas de “Aqui estão algumas sugestões” para “Atribuições sugeridas”.

* Não há alterações no modo como as atribuições inteligentes são calculadas para ocorrências. O nome da lista de atribuições inteligentes em uma lista de ocorrências é “Aqui estão algumas recomendações”.

Para obter mais informações, consulte [Visão geral de atribuições inteligentes](/help/quicksilver/manage-work/tasks/assign-tasks/smart-assignments.md).
