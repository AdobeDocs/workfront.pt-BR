Você pode visualizar todos os objetos disponíveis por meio da API do Adobe Workfront, juntamente com seus campos, no API Explorer. Para visualizar o API Explorer clique [aqui](https://developer.adobe.com/workfront/api-explorer/).

Para obter informações mais detalhadas sobre o uso do API Explorer, consulte [Uso do API Explorer](../../wf-api/general/using-api-explorer.md).
