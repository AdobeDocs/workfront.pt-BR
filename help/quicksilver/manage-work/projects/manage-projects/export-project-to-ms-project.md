# Exportar um projeto para o Projeto Microsoft

Você pode exportar projetos do Adobe Workfront para o Microsoft Project. 

>[!IMPORTANT]
>
>* Nem todos os campos do Workfront são transferidos no arquivo Projeto do Microsoft.\
   >  Para obter mais informações sobre a compatibilidade de campos entre o Workfront e o Microsoft Project, consulte o artigo [Mapear campos de projeto do Microsoft para projetos do Adobe Workfront](../../../manage-work/projects/manage-projects/map-ms-project-fields-to-workfront.md).
>* Recomendamos que você limite o número de vezes que transfere projetos de um aplicativo para outro. 
>

## Exportar um projeto do Workfront para o projeto do Microsoft

É possível exportar um projeto do Workfront da página do projeto ou de uma lista de projetos ou relatório.

1. Vá para o projeto que deseja exportar e clique no link **Mais** ícone ![](assets/qs-more-menu.png) à direita do nome do projeto

   ![](assets/project-level-more-drop-down-expanded-nwe-350x516.png)

   Ou

   Vá para uma lista de projetos ou relatório, selecione um projeto e clique no ícone Mais ![](assets/qs-more-menu.png) na parte superior da lista.

   ![](assets/more-menu-expanded-in-a-list-one-project-selected-nwe.png)

1. Clique em **Exportar projeto MS**.

   O projeto é baixado como um arquivo XML no seu computador e está pronto para ser importado para o Microsoft Project.
