# Use o [!UICONTROL Calendário doméstico] exibir

Você pode usar a exibição Calendário na [!UICONTROL Início] para gerenciar o dia de trabalho e as atribuições. Somente você pode visualizar e gerenciar seus [!UICONTROL Calendário doméstico].

>[!NOTE]
>
>O [!UICONTROL Calendário] no [!UICONTROL Início] é diferente de uma [!DNL Adobe Workfront] [!UICONTROL Calendário] , que é um relatório dinâmico que fornece uma representação visual do seu trabalho. Para obter informações sobre o [!DNL Workfront] [!UICONTROL Calendário] relatório, consulte [Visão geral dos relatórios de calendário](../../../reports-and-dashboards/reports/calendars/calendar-reports-overview.md).

## Agendar itens de trabalho na [!UICONTROL Calendário doméstico]

Você pode usar o [!UICONTROL Calendário doméstico] para programar itens de trabalho para a semana.

>[!NOTE]
>
>Colocar um item de trabalho em seu [!UICONTROL Calendário doméstico] não altera as datas existentes associadas ao item de trabalho.

Para adicionar uma [!DNL Workfront] item de trabalho do seu calendário:

1. Vá para o [!UICONTROL Calendário doméstico] , conforme explicado em [Visualize o [!UICONTROL Calendário doméstico]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).

   Todos os itens de trabalho atribuídos a você ou a seus grupos, equipes ou funções de trabalho são exibidos no painel esquerdo. Você pode classificar a lista e filtrá-la para exibir apenas determinados itens.

   Para obter mais informações, consulte [Filtrar e classificar itens de trabalho](#filter-and-sort-work-items) mais tarde neste artigo.

1. No painel esquerdo, arraste o item de trabalho para o dia e a hora em que deseja trabalhar nele.

   Se você se programar para trabalhar em um item após sua data de conclusão planejada ou por mais tempo do que o número de horas planejadas, o item de trabalho será exibido em vermelho e um aviso será exibido.

   Para saber mais sobre as datas de vencimento do item de trabalho e como elas afetam o tempo alocado na [!UICONTROL Calendário doméstico], consulte  [[!UICONTROL Calendário doméstico] exibir](../../../workfront-basics/using-home/using-the-home-area/home-calendar-view.md).

1. Siga um destes procedimentos:

   * **Para agendar um item de trabalho por vários dias:** Repita a etapa 2 para adicionar o item a cada dia e hora em que deseja trabalhar nele.
   * **Para modificar a duração de um item de calendário:** Passe o cursor do mouse sobre a borda inferior do item até que o cursor se torne uma seta dupla e arraste a borda inferior até o horário final desejado.
   * **Para excluir um item de calendário:** Passe o mouse sobre o item e clique no botão [!UICONTROL lixão] ícone que é exibido.

## Atualizar um item de trabalho em seu [!UICONTROL Calendário doméstico]

É possível fazer atualizações em itens de trabalho que são exibidos em [!UICONTROL Calendário doméstico] exibir. Suas atualizações também aparecem no [!UICONTROL Atualizações] da guia do item de trabalho real.

1. Vá para o [!UICONTROL Calendário doméstico] , conforme explicado em [Visualize o [!UICONTROL Calendário doméstico]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).

1. Abra o [!UICONTROL Detalhes] painel para o item de trabalho que deseja atualizar:

   * No painel esquerdo: Selecione o item de trabalho e clique em **[!UICONTROL Detalhes]**.
   * No calendário: Clique no evento de calendário agendado.

   Os detalhes do item de trabalho são exibidos em um painel que aparece à direita da janela.

   ![](assets/click-cal-item-to-see-details-350x217.png)

1. Siga um destes procedimentos para atualizar o item de trabalho:

   * No **[!UICONTROL Iniciar uma nova atualização]** digite a atualização, especifique qualquer uma das informações a seguir e clique em **[!UICONTROL Atualizar]**.

## Atualizar eventos de calendários integrados

Para atualizar os eventos de calendário integrados no [!UICONTROL Calendário doméstico]:

1. Vá para o [!UICONTROL Calendário doméstico] , conforme explicado em [Visualize o [!UICONTROL Calendário doméstico]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).
1. Clique em **[!UICONTROL Atualizar calendários]** no canto superior direito do [!UICONTROL Calendário doméstico].

   ![](assets/refresh-qs-350x360.png)

   Os eventos de calendário integrados são atualizados automaticamente ao abrir seu [!UICONTROL Calendário doméstico] visualizar ou atualizar [!UICONTROL Calendário doméstico].

## Navegar para uma semana diferente

1. Vá para o seu [!UICONTROL Calendário doméstico] , conforme explicado em [Visualize o [!UICONTROL Calendário doméstico]](../../../workfront-basics/using-home/using-the-home-area/view-home-calendar.md).
1. No **[!UICONTROL Barra de ferramentas do calendário]**, clique na seta para a esquerda do indicador de data para mover a exibição do calendário para trás uma semana ou para a direita para avançar uma semana.

   ![](assets/week-arrows-350x206.png)

   Você pode clicar em **[!UICONTROL Hoje]** para retornar à semana atual.

## Especificar quais calendários você visualiza

Você pode especificar quais calendários integrados, como seu PTO, Aniversário ou Calendário de feriados, deseja visualizar em seu [!UICONTROL Calendário doméstico]. Para obter instruções, consulte [Configure seu [!UICONTROL Calendário doméstico] configurações de exibição](../../../workfront-basics/using-home/using-the-home-area/configure-home-calendar-view.md).
