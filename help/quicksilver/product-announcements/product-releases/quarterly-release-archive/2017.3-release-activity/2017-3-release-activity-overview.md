## Aprimoramentos para dispositivos móveis {#mobile-enhancements}

>[!NOTE]
>
>A funcionalidade descrita nesta seção agora está disponível para uso com sua conta de Produção do Workfront; essa não é a funcionalidade Visualização.

A seguinte funcionalidade foi adicionada à versão de produção do aplicativo móvel do Workfront para Android e iOS em agosto de 2017:

* Enviar solicitações
* Nova Entrada na Planilha de Horas
* Edição de formulário personalizado
* Solicitações de aprovação de prova
* Notificações por push para itens inscritos

Para obter informações mais detalhadas sobre essas alterações, consulte [Aplicativo móvel atualizado para iOS e Android (início de agosto de 2017)](../../../announcements/announcement-archive/updated-mobile-app.md).

A seguinte funcionalidade adicional será disponibilizada antes da versão de Produção na semana de 8 de novembro:

* Acessar uma lista de projetos
* Acessar uma lista de tarefas e subtarefas em um projeto
* Acessar uma lista de problemas em um projeto
* Registrar um novo problema em um projeto
