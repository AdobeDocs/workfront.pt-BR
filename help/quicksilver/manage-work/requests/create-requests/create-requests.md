# Criar solicitações

<!--
{{highlighted-preview}}
-->

Solicitações e problemas são intercambiáveis no Adobe Workfront. Você pode criar solicitações seguindo um destes procedimentos:

* Enviar uma solicitação no Workfront
* Enviar uma solicitação em outro aplicativo
* Envie uma solicitação por email para um projeto configurado como Fila de solicitações.

Todas as solicitações são exibidas na área Problemas dos projetos que elas enviaram.

Para obter mais informações sobre como criar, enviar ou enviar solicitações de email no Workfront, consulte os seguintes artigos:
