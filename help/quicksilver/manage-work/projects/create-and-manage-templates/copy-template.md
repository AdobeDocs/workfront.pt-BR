# Copiar um modelo de projeto

Em vez de criar um novo modelo de projeto do zero, você pode copiar um modelo existente e fazer alterações, se necessário.

## Copiar um modelo

1. Vá para o modelo que deseja copiar.
1. Clique no botão **Mais** menu ![](assets/qs-more-icon-on-an-object.png), depois clique em **Copiar**.
1. Especifique um nome para o modelo no **Novo nome de modelo** campo.

   Por padrão, o novo nome é **Cópia de `<original template name>`.**

1. Selecione se deseja **Manter atribuições de usuário em tarefas e modelos**: selecione esta opção para transferir todas as atribuições de tarefa e modelo do modelo original para o novo modelo.
1. Clique em **Salvar** para criar uma cópia do template.
