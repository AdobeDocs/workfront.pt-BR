# Visão geral dos tipos de relatório do Painel de controle

>[!IMPORTANT]
>
>Esta página descreve a funcionalidade que está disponível como parte da versão beta do Canvas Dashboards. Consulte [Informações da versão beta dos Painéis de Tela](/help/quicksilver/product-announcements/betas/canvas-dashboards-beta/canvas-dashboards-beta-information.md) para saber mais.

Há vários relatórios diferentes que você pode adicionar a um painel do Canvas que visualizam seus dados do Workfront de diferentes maneiras. Os seguintes tipos de relatórios podem ser adicionados a um Painel de controle do Canvas:

* Relatórios existentes (relatórios criados usando a ferramenta de relatórios)
* Relatórios de gráficos (inclui gráficos de barras, colunas, linhas e pizza)
* Relatórios de KPI
* Relatórios de tabela
* Relatórios de aprovações pendentes
