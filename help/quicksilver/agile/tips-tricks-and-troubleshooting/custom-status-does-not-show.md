# Status personalizados não são exibidos no storyboard do ágil

Se quiser usar um status personalizado em um quadro de matérias ágil, o administrador do [!DNL Adobe Workfront] deve primeiro criar esse status personalizado como um status de todo o sistema. Para obter mais informações, peça ao administrador do [!DNL Workfront] para ver [Criar ou editar um status](../../administration-and-setup/customize-workfront/creating-custom-status-and-priority-labels/create-or-edit-a-status.md).
