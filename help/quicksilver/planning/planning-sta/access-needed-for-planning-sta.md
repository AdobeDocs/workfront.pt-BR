# Acesso necessário ao Adobe Workfront Planning como um produto independente

>[!IMPORTANT]
>
>As informações neste artigo se referem ao Adobe Workfront Planning, quando adquirido como um produto independente. Consulte este artigo quando sua empresa comprou um pacote somente do Adobe Workfront Planning e não comprou um pacote do Workfront Workflow.
>
>Para obter informações sobre o Adobe Workfront Planning quando adquirido junto com um pacote de Fluxo de Trabalho do Workfront, consulte [Introdução ao Adobe Workfront Planning](/help/quicksilver/planning/general/planning-overview.md).

Este artigo descreve as licenças, os níveis de acesso e os recursos do usuário para o Adobe Workfront Planning como um produto independente.

## Pacotes do Workfront Planning

Sua organização pode comprar um dos seguintes pacotes independentes do Workfront Planning:

* Prime
* Selecionar
* Ultimate

A tabela a seguir descreve o que está incluído em cada pacote:

| Recurso de planejamento | Selecionar | Prime | Ultimate |
|---|---|---|---|
| Planejamento, orquestração e gerenciamento de campanhas | ✓ | ✓ | ✓ |
| Espaços de trabalho limitados | ✓ | ✓ | ✓ |
| Registros por espaço de trabalho | 25,000 | 500,000 | 1,000,000 |
| Total de registros (todos os espaços de trabalho) | 500,000 | 2,000,000 | Ilimitado |
| Tipos de Registro Global e conexões entre espaços de trabalho | — | ✓ | ✓ |
| Acesso a recursos futuros no lançamento | Alguns | Alguns | Mais |

## Disponibilidade do pacote do Workfront Planning

<!--
the bullets repeat in the "Planning overview" article
-->

O Workfront Planning é acessível quando sua organização compra um dos seguintes pacotes Workfront:

* Workfront Workflow e Workfront Planning comprados juntos. Todos os usuários na organização têm uma licença do Workflow e do Planning. Isso dá a todos os usuários acesso total a todos os recursos do Workfront para ambos os módulos.

* O Workfront Workflow para todos em sua organização e o Workfront Planning somente para alguns usuários em sua organização. Isso dá aos usuários acesso total a todos os recursos do Fluxo de Trabalho e um acesso mais limitado aos recursos do Planning para os usuários que recebem uma licença do Planning.

* O Workfront Planning como um produto independente para usuários em sua organização. Isso não dá aos usuários acesso a nenhum recurso do Workfront Workflow e acesso aos recursos do Planning.

Para obter informações sobre recursos incluídos no Planning como um produto independente, consulte a seção &quot;Funcionalidade incluída no Workfront Planning como um produto independente&quot; no artigo [Introdução ao Adobe Workfront Planning como um produto independente](/help/quicksilver/planning/planning-sta/planning-sta-overview.md).

## Licença de usuário e níveis de acesso

<!-- should this be moved to the Manage users in Planning article??-->

As licenças do Planning fazem parte do nível de acesso atribuído aos usuários.

Para o Planning como um produto independente, não é possível atribuir licenças aos níveis de acesso - eles já estão codificados para serem incluídos nos níveis de acesso atribuídos aos usuários.

Dependendo da combinação de pacotes do Workfront Planning que sua organização comprar, você poderá ter um dos seguintes níveis de acesso para os usuários:

* **Administrador do Planning**: Disponível para pacotes do Workfront Planning. Os administradores do Planning são atribuídos automaticamente quando os usuários são adicionados ao Admin Console.
* **Planning Standard**: disponível para todos os pacotes do Workfront Planning. Você pode atribuir esses níveis de acesso aos usuários ao criá-los.

### Visão geral de navegação do Administrador do Planning

<!--
Managing your Adobe Workfront Planning instance is similar to managing an Adobe Workfront with Planning instance, but there are some differences.

Um usuário com um nível de acesso de Administrador do Planning tem os seguintes recursos:

* Tem acesso de gerenciamento total ao sistema.
* Pode criar e editar usuários, gerenciar equipes, configurar trimestres personalizados e acessar todas as subpáginas de Configuração.

  Para obter informações, consulte:

   * [Gerenciar usuários no Adobe Workfront Planning como um produto independente](/help/quicksilver/planning/planning-sta/manage-users-in-planning-sta.md)
   * [Gerenciar equipes no Adobe Workfront Planning como um produto independente](/help/quicksilver/planning/planning-sta/manage-teams-in-planning-sta.md)
* Pode enviar e gerenciar solicitações.

  Para obter informações, consulte [Solicitações do Adobe Workfront Planning: índice do artigo](/help/quicksilver/planning/requests/requests-article-index.md).
* Tem as seguintes áreas no menu principal:

   * **Planning**: tem recursos completos para objetos do Planning, para criar, excluir, compartilhar e conectá-los.
   * **Usuários**: você pode adicionar usuários e editar seus perfis.
   * **Solicitações**
   * **Configuração**
* Tem as seguintes seções na área Configuração:

   * **Equipes**: você pode adicionar, remover ou editar equipes. A edição é limitada ao nome da equipe, à descrição e aos membros; nenhum controle de filtro, exibição, agrupamento ou exportação está disponível.
   * **Fazer Logon como**: representar outro usuário para fins de solução de problemas.
   * **Trimestres Personalizados**: Configure trimestres fiscais personalizados que aparecem nas exibições de linha do tempo do Planning.
   * Sistema

* Possui as seguintes seções na área Sistema:

   * **Informações do cliente**: exibir detalhes do cliente e da organização.
   * **Preferências**: revise e configure as preferências de nível de sistema.

### Visão geral de navegação do Planning Standard

Um usuário com um nível de acesso do Planning Standard tem os seguintes recursos:

* Pode gerenciar espaços de trabalho e seu conteúdo.
* Pode enviar e gerenciar solicitações.

  Para obter informações, consulte [Solicitações do Adobe Workfront Planning: índice do artigo](/help/quicksilver/planning/requests/requests-article-index.md).
* Os usuários do Planning Standard podem acessar as seguintes áreas no menu Principal:

   * **Planejamento**
   * **Usuários**: eles têm acesso somente para visualização aos usuários. Eles não podem criar ou editar usuários. <!--not sure if this is still true-->
   * **Solicitações**

* Não tem acesso à Configuração nem a nenhuma de suas seções.

## Configurar Níveis de Acesso para o Planning como um produto independente

Os níveis de acesso são incorporados quando você compra o Planning como um produto independente e não pode configurá-los para os usuários do Planning.

Para atribuir níveis de acesso aos usuários, como Administrador do Planning:

* Crie usuários no Console do Adobe.

  Existem os seguintes cenários:

   * Os usuários adicionados à Console do Adobe, à medida que os Administradores recebem um nível de acesso de Administrador do Planning no Workfront Planning.
   * Os usuários adicionados ao Console do Adobe podem receber um nível de acesso do Planning Standard no Workfront Planning. Esse é o único acesso disponível para atribuir a novos usuários no Workfront Planning como um produto independente.

Para obter mais informações, consulte [Gerenciar usuários](/help/quicksilver/planning/planning-sta/manage-users-in-planning-sta.md).

## Conceder permissões no Workfront Planning como um produto independente

Você pode compartilhar os seguintes objetos no Workfront Planning como um produto independente:

* Espaços de trabalho
* Tipos de registro
* Registros
* Exibições

O compartilhamento de objetos no Planning como um produto independente é idêntico ao compartilhamento deles no Planning quando adquiridos junto com um pacote de Fluxo de trabalho.

Para obter mais informações, consulte [informações de acesso do Adobe Workfront Planning: índice do artigo](/help/quicksilver/planning/access/access-information.md).
