# Visão geral da atividade da versão 2019.1

Esta página fornece informações sobre a funcionalidade incluída na versão 2019.1.

>[!IMPORTANT]
>
> A funcionalidade no ambiente de Visualização está sujeita a alterações antes do lançamento para o ambiente de Produção. Se estiver visualizando uma versão do PDF deste documento, acesse a versão online para obter as informações mais atualizadas (os links deste documento vão para a versão online).

A versão 2019.1 contém aprimoramentos para administradores e outros usuários.

## Aprimoramentos para administradores

* [Restaurar modelos da Lixeira](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#restore)

   Os administradores do Workfront agora podem restaurar modelos da Lixeira.

* [Mostrar carimbos de data e hora para campos de data na página inicial](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#show)

   Os administradores do Workfront agora podem optar por mostrar ou ocultar carimbos de data e hora para datas de vencimento na Lista de trabalho e no Calendário usando Modelos de layout.

* [Configurar a função de prova padrão para não destinatários que abrem uma prova](https://support.workfront.com/hc/en-us/articles/360016372633#configure-default-proofing-role)

   Os administradores do Workfront agora podem configurar a função de prova padrão para usuários que não são recipients designados no fluxo de trabalho da prova, mas que têm acesso à prova por meio de seu objeto Workfront (como projeto, tarefa ou problema).

* [Todos os tipos de duração disponíveis em Preferências de projeto](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#all)

   Ao definir o tipo de tarefa padrão e duração da emissão na área Preferências do projeto em Configurar, os administradores do Workfront podem usar as opções Simples, Orientado pelo Esforço, Atribuição Calculada e Trabalho Calculado.

## Melhorias para todos os usuários

### Melhorias do projeto

* [Melhorias no ágil](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#agile)

   Veja as informações necessárias filtrando o quadro Kanban ou Scrum por usuário individual ou por equipe.

* [Adicionar tarefas e problemas de uma lista a uma iteração](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#add)

   Agora é possível usar uma tarefa ou lista de problemas, um relatório ou um painel para adicionar histórias a uma iteração. Você também pode ter várias equipes atribuídas histórias em uma iteração.

* [Novos tamanhos de papel para exportação de gráfico de Gantt](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#new)

   Agora é possível imprimir o gráfico de Gantt nos tamanhos de papel A1 e A2.

* [Remoção do acesso às caixas de diálogo no gráfico de Gantt no modo de edição](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#removed)

   Não é mais possível acessar a caixa de diálogo Atribuições avançadas enquanto o gráfico de Gantt está no modo de edição. Você só pode fazer edições em linha enquanto a Lista de tarefas do Gráfico de Gantt está no modo de edição.

* [Exibir Informações da Tarefa no Gráfico de Gantt da Lista de Projetos em um Programa ou Portfolio](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#view)

   Agora é possível visualizar informações sobre as tarefas de um projeto no Gráfico de Gantt da Lista de Projetos em um Programa ou Portfolio.

* [Acessar o Gráfico de Gantt da Lista de Tarefas em Modelos](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#access)

   Agora você pode acessar o Gráfico de Gantt da Lista de Tarefas em um modelo de projeto.

* [Exibição de projeto renomeada no gráfico de Gantt](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#renamed)

   Renomeamos a opção de exibição &quot;Projeto&quot; no gráfico de Gantt para &quot;Ajustar tudo&quot;.

* [Cenários de hipóteses no Gráfico de Gantt da lista de tarefas](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#what-if)

   Agora é possível executar várias ações nas tarefas em um projeto quando elas são exibidas no modo de edição do gráfico de Gantt.

* [Melhorias na lista de tarefas](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#task)

   As listas de tarefas recém-reprojetadas foram reativadas, após uma breve remoção durante a versão 18.3 beta 4. Vários recursos adicionais também estão disponíveis.

* [Exibir listas na tela cheia](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#display)

   Quando uma lista de projetos ou tarefas é maior que o tamanho da sua tela, a lista agora é exibida automaticamente em toda a janela do navegador à medida que você rolar.

* [Novas listas em áreas adicionais](../../../../product-announcements/product-releases/quarterly-release-archive/2019.1-release-activity/2019.1-project-enhancements.md#new2)

   Melhoramos o desempenho e aprimoramos a aparência das listas de projetos e tarefas em áreas adicionais.

* [Enviar um Relatório Entregue no Formato XLSX](https://support.workfront.com/hc/en-us/articles/360016246394#send-a-delivered-report)

   Agora você pode agendar um relatório para ser entregue no formato MS Excel (.xlsx), além de todos os outros formatos atuais.

* [Exportar logs de auditoria](https://support.workfront.com/hc/en-us/articles/360016246394#export-audit-logs)

   Agora é possível exportar entradas de log de auditoria para um arquivo CSV. Você pode exportar até 50.000 entradas de log de auditoria para um arquivo CSV de uma vez.

* [A exibição WBS em uma lista de tarefas não está mais disponível clicando com o botão direito do mouse na coluna Número da tarefa](https://support.workfront.com/hc/en-us/articles/360016246394#wbs-view-from-the-task-list)

   Não é mais possível clicar com o botão direito do mouse na coluna Número da tarefa em uma lista de tarefas para mostrar a Estrutura de Detalhamento do Trabalho (WBS) do projeto.

* [Linha do tempo do agendamento persistente](https://support.workfront.com/hc/en-us/articles/360016247594#persistent-scheduling-timeline)

   Quando você não aplica filtros baseados no usuário ao Planejador de Recursos, todos os usuários do sistema agora são exibidos na Visualização de Usuário do Planejador de Recursos.

### Aprimoramentos de revisão de texto

* [Prova de conteúdo interativo no visualizador de prova web](https://support.workfront.com/hc/en-us/articles/360016372633#proof-interactive-content-in-the-web-proofing-viewer)

   Se as políticas de segurança da sua organização não permitirem o uso do aplicativo independente Desktop Proofing Viewer, o administrador do Workfront poderá ativar o conteúdo interativo no Web Proofing Viewer.

* [A Ordem de classificação padrão para comentários no visualizador de prova agora é da mais antiga para a mais recente](https://support.workfront.com/hc/en-us/articles/360016372633#default-sorting-order-for-comments)

   No visualizador de prova, a ordem de classificação padrão para comentários em uma prova agora é do Mais antigo para o Mais recente, como em uma conversa verbal.

* [Revisão aprimorada de comentários no visualizador de prova associado a um intervalo de vídeo](https://support.workfront.com/hc/en-us/articles/360016372633#enhanced-reviewing-for-comments)

   Ao revisar um comentário associado a uma variedade de vídeos no visualizador de prova, você pode clicar em Reproduzir para exibir toda a gama de imagens.

* [Link para Detalhes do documento a partir de uma notificação de prova ou do visualizador de prova](https://support.workfront.com/hc/en-us/articles/360016372633#link-to-document-details)

   Ao receber um email convidando-o para revisar uma prova ou ao revisar uma prova no visualizador de prova, agora é possível acessar rapidamente a página Detalhes do documento para a prova.

* [Alterar suas notificações por email no visualizador de prova](https://support.workfront.com/hc/en-us/articles/360016372633#change-your-email-notifications)

   Agora, todos os revisores de prova podem especificar quais notificações de prova desejam receber para uma prova. Isso é particularmente importante ao colaborar com partes interessadas externas.

* [Alterar a cor do fundo no visualizador de prova de desktop](https://support.workfront.com/hc/en-us/articles/360016372633#change-the-background-color-in-the-desktop-proofing-viewer)

   Agora você pode alterar a cor de fundo do Desktop Proofing Viewer do padrão quase preto para branco.

* [Limpar dados do navegador em cache de uma prova](https://support.workfront.com/hc/en-us/articles/360016372633#clear-cached-browser-data-from-a-proof)

   Agora, é possível limpar os dados de cache do navegador que podem ser salvos com uma prova para que todo o conteúdo apareça no Visualizador de prova de desktop e os revisores possam ver e comentar sobre ele.

### A versão 4 da API não é mais suportada

O suporte para a API versão 4 está terminando em conjunto com a versão 2019.1. Para obter mais informações sobre o cronograma de suporte para versões da API do Workfront, consulte  [Controle de versão da API e cronograma de suporte](../../../../wf-api/api/api-version-support-schedule.md).

### Atualizações da ferramenta ágil

As seguintes atualizações para a ferramenta Ágil foram disponibilizadas diretamente no ambiente de Produção em 13 de dezembro de 2018:

* Itens de Filtro e Grupo na guia de backlog de Kanban
* Exibir informações do pai para tarefas e problemas em uma iteração

### A API versão 10 já está disponível na Visualização {#api-version-10-now-available-in-preview}

A API versão 10 já está disponível no ambiente de Visualização. Ele será disponibilizado no ambiente de Produção com a versão 18.3. Para obter mais informações, consulte [Novidades na API versão 10.](https://support.workfront.com/hc/en-us/articles/360010455333) 

### Várias opções de logon único a serem removidas {#various-single-sign-on-options-to-be-removed}

As seguintes opções de Logon único (SSO) agora são suportadas apenas em uma capacidade limitada; serão retirados do produto no segundo semestre de 2019:

* LDAP\
   Para obter mais informações sobre como usar o LDAP com o Workfront, consulte [Configurando o Workfront com o LDAP.](https://support.workfront.com/hc/en-us/articles/217223517)

* Active Directory\
   Para obter mais informações sobre o uso do Ative Diretory com o Workfront, consulte [Configurando o Workfront com o Ative Diretory.](https://support.workfront.com/hc/en-us/articles/217223547)

* SAML 1.1\
   Para obter mais informações sobre o uso do SAML 1.1 com Workfront, consulte [Configuração do Workfront com SAML 1.1.](https://support.workfront.com/hc/en-us/articles/217194887)

A seguir, a linha do tempo de remoção para essas opções de SSO:

* **Suporte limitado:** Agosto de 2018 - Janeiro de 2019\
   As principais questões serão corrigidas; problemas de baixa gravidade não serão resolvidos.

* **Substituição:** Janeiro de 2019 - agosto de 2019\
   Não suportado.

* **Não está mais disponível no Workfront:** Agosto de 2019

Essas alterações estão sendo feitas em preparação para a experiência de autenticação aprimorada, que atualmente está disponível somente para um número limitado de clientes. Para obter mais informações sobre essas alterações futuras, consulte [Introdução à experiência de autenticação aprimorada.](https://support.workfront.com/hc/en-us/articles/360008329033)
