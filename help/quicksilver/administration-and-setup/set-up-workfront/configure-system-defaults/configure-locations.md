# Configurar locais

{{highlighted-preview-article-level}}

Os cartões de taxa permitem que sua organização gerencie facilmente as taxas de cobrança dos projetos. Para obter mais informações, consulte [Gerenciar cartões de taxa](/help/quicksilver/administration-and-setup/set-up-workfront/configure-system-defaults/manage-rate-cards.md). Como um [!DNL Adobe] [!DNL Workfront] administrador, você pode configurar os locais padrão disponíveis para atribuir funções de trabalho em cartões de taxa. Isso garante que os cartões de taxa reflitam com precisão as taxas de mercado em cada local.

## Configurar locais para toda a organização

{{step-1-to-setup}}

1. No painel esquerdo, clique em [!UICONTROL **Localizações**].
1. Clique em [!UICONTROL **Adicionar mais locais**].
1. Insira o nome e a descrição do local.

   O nome da localização deve começar com uma letra maiúscula e as localizações não podem ser duplicadas.

1. Clique fora da área de entrada para salvar o local.
1. Para excluir um local, selecione-o na lista e clique no link **Excluir** ícone ![Ícone Excluir](assets/delete.png).

>[!NOTE]
>
>Os locais associados às funções de trabalho em um cartão de taxa não podem ser excluídos.
