# Gerenciar informações de trabalho

Você pode usar o Adobe Workfront para gerenciar seu trabalho.

Para começar, determine quais objetos de trabalho você usa para definir seu trabalho, dependendo da complexidade e dos recursos ou orçamento que você aloca para ele.

Esta seção descreve como organizar seu trabalho em itens maiores, como projetos, portfólios ou programas, e como você pode dividir grandes quantidades de trabalho em itens mais gerenciáveis, como tarefas e problemas.

Além disso, você pode usar o gráfico de Gantt para definir uma linha do tempo do seu trabalho ou delegá-lo a outras pessoas quando não estiver disponível para concluí-lo.
