# Criar relatórios diretos

Você pode designar usuários como relatórios diretos para outro usuário. Isso permite criar um gráfico de organização que exibe a estrutura de gerenciamento de sua organização. Para obter mais informações, consulte [Exibir o gráfico organizacional](../../../people-teams-and-groups/work-directly-with-others/view-the-org-chart.md).

## Criar relatórios diretos

1. Comece a editar um usuário conforme descrito em [Editar o perfil de um usuário](../../../administration-and-setup/add-users/create-and-manage-users/edit-a-users-profile.md).
1. No **Organização** certifique-se de que uma empresa esteja selecionada na seção **Empresa** campo.

   Este campo não pode estar em branco.

1. No **Relatórios diretos** , especifique quaisquer usuários que relatam para o usuário que você está editando.

1. (Opcional) Na seção **Relatórios para** , especifique o nome do usuário para o qual esse usuário relata.
1. Clique em **Salvar alterações**.
