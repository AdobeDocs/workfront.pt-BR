# Instalar o Adobe Cloud Drive

Para trabalhar com os projetos de armazenamento em nuvem do Adobe no desktop, instale o Adobe Cloud Drive no computador Mac ou Windows.

Para obter informações sobre o que é o Adobe Cloud Drive e o que você pode fazer com ele, consulte [visão geral do Adobe Cloud Drive](/help/quicksilver/documents/adobe-cloud-drive/adobe-cloud-drive-overview.md).

## Pré-requisitos

Antes de instalar o Adobe Cloud Drive, verifique o seguinte:

* Seu dispositivo atende aos requisitos de sistema listados na seção [Requisitos de sistema](#system-requirements) deste artigo.
* Sua organização possui uma versão do Workfront compatível com o armazenamento em nuvem da Adobe.

Se sua organização gerencia instalações de aplicativos para você e você não vir o Adobe Cloud Drive disponível para download, entre em contato com o administrador do Workfront.

## Requisitos do sistema

**Sistemas operacionais:**

* macOS 13.5 ou mais recente (Intel e Apple Silicon)
* Windows 11 22H2 (22621.6060) ou mais recente (x64)
* Windows 11 23H2 (22631.6199) ou mais recente (ARM)

**Hardware:**

* Mínimo de 4 GB RAM
* 500 MB de espaço livre em disco para o aplicativo
* Espaço adicional para arquivos em cache (varia de acordo com o uso)

**Rede:**

* Conexão com a Internet para configuração e sincronização iniciais
* Conexão estável recomendada para desempenho

Para garantir que o Adobe Cloud Drive possa acessar os serviços dos quais depende, permita os pontos de extremidade de rede listados em [pontos de extremidade de rede do Adobe](https://helpx.adobe.com/in/enterprise/kb/network-endpoints.html).

## Baixar o instalador

1. Vá para [Adobe Software Distribution](https://experience.adobe.com/#/downloads).

1. Faça logon com sua Adobe ID.
1. Na lista de downloads **Geral**, localize a **Adobe Cloud Drive**.
1. Selecione o instalador do seu sistema operacional:

   * **macOS:** Adobe Cloud Drive para Mac (ARM64 ou Intel)
   * **Windows:** Adobe Cloud Drive para Windows (x64 ou ARM64)

1. Clique em **Baixar**.

   >[!IMPORTANT]
   >
   >Caso não veja o instalador do Adobe Cloud Drive, sua organização pode não estar em uma versão do Workfront compatível com o armazenamento na nuvem da Adobe. Entre em contato com o administrador do Workfront.

## Instalar o Adobe Cloud Drive

Use o procedimento para o seu sistema operacional.

### Instalar no macOS

1. Abra o arquivo `.dmg` baixado.
1. Clique duas vezes no ícone de instalação exibido.
1. Siga o assistente de configuração na tela.

1. Quando solicitado, faça logon com sua Adobe ID.

   Se outro aplicativo de desktop do Adobe já estiver em execução e você estiver conectado, o Adobe Cloud Drive usará essa sessão ativa automaticamente.

### Instalar no Windows

1. Execute o instalador `.exe` baixado.

1. Siga os prompts no assistente de instalação.
1. Clique em **Instalar** quando solicitado.
1. Inicie a **Adobe Cloud Drive** pelo menu Iniciar.
1. Quando solicitado, faça logon com sua Adobe ID.

## Verificar a instalação

Após a instalação, o Adobe Cloud Drive é exibido como uma unidade montada no computador:

* **macOS:** na barra lateral do Localizador em **Locais**, como **Adobe Cloud Drive**.
* **Windows:** No Explorador de Arquivos em **Este PC**, como **Adobe Cloud Drive**.

Você deve ver seus projetos de armazenamento em nuvem do Adobe listados como pastas dentro da unidade.

Para obter informações sobre como trabalhar com seus projetos na unidade, consulte [Usar a Adobe Cloud Drive](/help/quicksilver/documents/adobe-cloud-drive/use-adobe-cloud-drive.md).
