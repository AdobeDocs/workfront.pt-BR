>[!IMPORTANT]
>
>[O Microsoft desabilitou o suporte para tokens herdados do Exchange online](https://learn.microsoft.com/pt-br/office/dev/add-ins/outlook/faq-nested-app-auth-outlook-legacy-tokens), que foram usados pelo suplemento do Workfront Outlook para autenticação. >
>**Como o Microsoft desabilitou esses tokens, a integração do Workfront para Microsoft Outlook não está mais disponível.**
