# Definir prazos individuais do revisor

É possível definir diferentes prazos de conclusão para diferentes revisores em uma prova já criada. Isso requer a criação de um estágio separado para cada revisor para o qual você deseja definir um prazo individual.

## Definir prazos individuais do revisor

1. Passe o mouse sobre a prova e clique em **Fluxo de trabalho de revisão** para abrir a página Fluxo de trabalho.
1. No canto superior direito, clique em **Nova etapa**

   ![Novo_Botão_de_Estágio.png](assets/new-stage-button.png)

   >[!NOTE]
   >
   >Se você não vir a nova opção de estágio, primeiro deve converter o fluxo de trabalho em um fluxo de trabalho automatizado.

1. Adicione informações para o estágio, especialmente o nome do revisor e o prazo desejado para o revisor.
1. Repita as duas etapas anteriores, conforme necessário, para adicionar outros revisores que precisam de seu próprio prazo.
