# Lista de blueprints disponíveis

O seguinte [!DNL Adobe Workfront] os blueprints estão disponíveis para que os administradores do sistema instalem. Para obter mais informações sobre blueprints, consulte [Visão geral do Blueprints](/help/quicksilver/administration-and-setup/blueprints/blueprints-overview.md).

**[!UICONTROL Adicionar nova lista de verificação de TI de funcionário]**
Este blueprint contém um modelo da variável [!DNL Workfront] a organização de serviços profissionais tem utilizado, durante anos, os contratos de implementação com as equipes corporativas de TI para organizar novas atividades de integração de funcionários. O uso desse modelo permite que as equipes de TI operem com eficiência, o que leva a uma nova experiência positiva para os funcionários e a um controle mais rápido da produtividade.

**[!UICONTROL Planeamento Orçamental Anual]**
Este blueprint contém um modelo da variável [!DNL Workfront] a organização de serviços profissionais tem utilizado durante anos compromissos de implementação com as equipes das finanças para organizar as atividades relacionadas com o planeamento orçamental anual. O modelo ajuda a definir expectativas com os líderes da unidade de negócios e os orienta por meio do processo para solicitar e reservar orçamentos durante o processo de planejamento anual.

**[!UICONTROL Plano de Entrega de Serviços Contratados]**
Este blueprint contém um template que organiza o trabalho relacionado ao fornecimento de serviços voltados para o cliente. Ele foi projetado para criar uma abordagem uniforme para a entrega de serviços contratados.

**[!UICONTROL Plano de evento corporativo]**
Este blueprint contém um modelo da variável [!DNL Workfront] a organização de serviços profissionais tem utilizado, durante anos, os contratos de implementação com as equipes de Recursos Humanos para apoiar as atividades de retenção de funcionários, especificamente eventos e partes corporativas.

**[!UICONTROL Desativar Lista de Verificação de TI de Funcionário Terminado]**
Este blueprint contém um modelo da variável [!DNL Workfront] a organização de serviços profissionais tem utilizado durante anos os contratos de implementação com equipes corporativas de TI para gerenciar as várias atividades que devem ocorrer quando um funcionário encerra seu emprego. É essencial desativar o acesso dos funcionários e fazer outras alterações nos sistemas para reduzir a exposição ao risco e garantir que outros membros da equipe possam permanecer produtivos na ausência do funcionário que sai.

**[!UICONTROL Produção de ativos digitais] (Integrado)**
O [!UICONTROL Produção de ativos digitais] o blueprint para processos integrados contém um modelo que organiza o trabalho para um processo de produção digital padrão. O modelo fornece um processo simples para a criação original de ativos digitais de uma equipe que depende de procedimentos operacionais padrão para definir como fazer o trabalho e usa o projeto apenas para priorizar o trabalho e rastrear o progresso.

**[!UICONTROL Produção de ativos digitais] (Gerenciado)**
O [!UICONTROL Produção de ativos digitais] contém um template que organiza o trabalho para um processo de produção digital padrão. Ela serve como um ponto de entrada para gerenciar a criação de ativos digitais no Workfront. Use este pacote para criar uma base para a criação de conteúdo digital e, em seguida, personalizá-lo para atender às necessidades de outros processos criativos para materiais digitais.

**[!UICONTROL Revisão de ativos digitais] (Integrado)**
O [!UICONTROL Revisão de ativos digitais] o blueprint de um processo integrado contém um único modelo para organizar o processo de trabalho associado à revisão de um ativo existente. O modelo fornece um processo simples para uma equipe que depende de procedimentos operacionais padrão para definir como fazer o trabalho e usa o projeto apenas para priorizar o trabalho e rastrear o progresso.

**[!UICONTROL Revisão de ativos digitais] (Gerenciado)**
O [!UICONTROL Revisão de ativos digitais] O pacote contém um único modelo para organizar o trabalho associado a uma revisão para um ativo existente. O modelo fornece um processo simplificado para uma revisão de ativos, enquanto cria uma experiência consistente em comparação a um novo projeto de produção de ativos.

**[!UICONTROL Editar Lista de Verificação de TI do Funcionário Transferente]**
Este blueprint contém um modelo da variável [!DNL Workfront] a organização de serviços profissionais tem utilizado, durante anos, os contratos de implementação com as equipes corporativas de TI para gerenciar as várias atividades que devem ocorrer quando um funcionário transfere para uma nova função. Este modelo reduz os riscos de segurança que existem quando uma transferência de funcionários é concluída sem supervisão de TI. Também elimina o desperdício de horas de tempo improdutivo, quando o acesso aos sistemas não é concedido para cumprir os deveres do novo papel do funcionário.

**[!UICONTROL Fim do Mês de Fechamento Operacional]**

O modelo fornece uma lista de verificação de atividades comuns que devem ser concluídas mensalmente para relatar e promover com precisão a integridade dos negócios.
