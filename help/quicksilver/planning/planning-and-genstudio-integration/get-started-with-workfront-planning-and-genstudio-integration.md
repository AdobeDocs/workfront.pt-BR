Há uma integração nativa entre o Adobe GenStudio for Performance Marketing e o Adobe Workfront Planning. A finalidade dessa integração é evitar que os usuários precisem alternar entre os dois aplicativos para atualizar informações idênticas ou semelhantes.

Agora você pode atualizar ou criar suas campanhas no GenStudio ou no Workfront Planning e continuar trabalhando nelas a partir de qualquer aplicativo.

O espaço de trabalho do GenStudio for Performance Marketing estará disponível no Adobe Workfront Planning quando sua empresa tiver comprado ambos os produtos.

Com a integração entre o Workfront Planning e o GenStudio for Performance Marketing, você pode:

<!--check this list and ensure it's accurate and add/ remove some of the benefits-->

* Exiba o espaço de trabalho do GenStudio no Workfront Planning.
* Modifique suas campanhas no GenStudio e atualize em tempo real as mesmas informações no Workfront Planning.
* Modifique suas campanhas no Workfront Planning e atualize em tempo real as mesmas informações no GenStudio.

## Requisitos de integração

* O Workfront e o GenStudio for Performance Marketing devem ser habilitados para a mesma organização. Se sua empresa tiver várias instâncias do Workfront, somente uma poderá ser integrada ao GenStudio for Performance Marketing.

* A instância do Workfront faz parte da Experiência unificada do Adobe, incluindo o uso do Identity Management System (IMS).

  Para obter informações, consulte [Experiência unificada da Adobe para Workfront](/help/quicksilver/workfront-basics/navigate-workfront/workfront-navigation/adobe-unified-experience.md).

* Os usuários que usam tanto o Planning quanto o GenStudio podem pertencer a apenas uma instância do Workfront na organização IMS.

## Considerações para o gerenciamento de um espaço de trabalho do GenStudio no Workfront Planning

* Sua organização deve comprar o Adobe GenStudio for Performance Marketing antes de exibir um espaço de trabalho do GenStudio no Workfront Planning.

  Para obter mais informações sobre o GenStudio, consulte [Guia do Usuário do Adobe GenStudio for Performance Marketing](https://experienceleague.adobe.com/pt-br/docs/genstudio-for-performance-marketing/user-guide/home).

  Para obter mais informações sobre a integração do GenStudio e do Workfront Planning, consulte [Introdução à integração do Workfront Planning e do GenStudio for Performance Marketing](/help/quicksilver/planning/planning-and-genstudio-integration/get-started-with-workfront-planning-and-genstudio-integration.md)

* Os usuários do Workfront devem ter acesso ao GenStudio para visualizar o espaço de trabalho do GenStudio no Workfront Planning.


* As seções a seguir descrevem algumas limitações para o que você pode ou não gerenciar em um espaço de trabalho do GenStudio no Workfront Planning.

### O espaço de trabalho do GenStudio no Workfront Planning

* Se sua organização tiver várias instâncias do Workfront, somente uma instância do Workfront poderá exibir seu espaço de trabalho do GenStudio.
* O espaço de trabalho do GenStudio exibe um indicador visual que deixa claro que ele é importado do GenStudio.

### Tipos de registro

* Não é possível editar tipos de registro do GenStudio no Workfront Planning.
* Não é possível compartilhar tipos de registro do GenStudio com outros usuários. Os administradores do Workfront podem exibir o espaço de trabalho do GenStudio em sua área de Planejamento.
* Os tipos de registro sincronizados com o GenStudio exibem um indicador visual que deixa claro que os tipos de registro são importados do GenStudio.

### Registros

* É possível adicionar ou excluir registros no GenStudio e eles se tornam visíveis no (ou são removidos do) Workfront Planning.
É possível adicionar ou excluir registros no Workfront Planning e eles se tornam visíveis no (ou são removidos do) GenStudio.
* Você pode adicionar registros do Workfront Planning das seguintes maneiras:

   * Manualmente, do zero
   * Importando-os usando um arquivo CSV ou do Excel

  Para obter informações, consulte [Criar registros](/help/quicksilver/planning/records/create-records.md).
* Não é possível criar ou excluir registros de Ativação do Workfront Planning.

### Campos

* Todos os campos de registro são importados do GenStudio e as configurações de campo não podem ser editadas.
* Você pode criar campos para tipos de registro do GenStudio no Workfront Planning somente se tiver acesso de administrador do sistema no GenStudio.
* É possível editar informações de registro em todos os registros no espaço de trabalho do GenStudio em qualquer um dos campos visíveis do Workfront Planning.

  Para obter informações, consulte [Editar registros](/help/quicksilver/planning/records/edit-records.md).
* Você pode ocultar campos na exibição de tabela de um tipo de registro do GenStudio, mas não pode excluir campos do Workfront Planning.

### Visualizações

* É possível criar exibições para tipos de registro do GenStudio. Não é possível editar exibições automaticamente importadas do GenStudio, mas você pode alterar os elementos de exibição para a exibição de tabela do GenStudio. Por exemplo, você pode modificar os filtros, a classificação, os agrupamentos, as cores das linhas e a altura das linhas na exibição da tabela.

  Para obter informações, consulte [Gerenciar exibições de registros](/help/quicksilver/planning/views/manage-record-views.md).

* Você pode compartilhar a exibição de um tipo de registro do GenStudio das seguintes maneiras:

   * Copiar o link de visualização
   * Exportar a exibição para um arquivo (disponível somente para a exibição de tabela)

### Conexões

* Não é possível conectar outros tipos de registro ou objeto a partir de tipos de registro do GenStudio no Planning.
* Você pode se conectar a tipos de registro do GenStudio a partir de outros tipos de registro no Planning.
