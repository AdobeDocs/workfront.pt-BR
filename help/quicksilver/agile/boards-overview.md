# Visão geral dos [!UICONTROL painéis]

[!DNL Adobe Workfront] Os [!UICONTROL painéis] são ferramentas flexíveis que permitem a colaboração em equipe, fornecendo acesso a um painel compartilhado que contém colunas e cartões.

Mostrar o status das tarefas é um uso comum para painéis, mas você pode agrupar qualquer item. Um cartão em um painel pode representar uma pessoa, grupo de pessoas, ideia, conceito ou qualquer tipo de item de que você precisa.

Ao usar painéis, você pode:

* Configurar rapidamente um painel de tarefas
* Configurar colunas para mostrar um status ou categoria
* Personalizar fluxos de trabalho com cartões abertos (cartões ad hoc ou cartões conectados a tarefas e problemas do [!DNL Workfront])
* Organizar e gerenciar trabalhos para uma equipe em um só lugar
* Usar ferramentas de planejamento ágeis, como uma coluna de entrada para placas Kanban

Para obter informações sobre como usar os recursos básicos de painéis, consulte [Introdução aos painéis: índice de artigo](../agile/get-started-with-boards/get-started-with-boards.md). Para obter informações sobre as ferramentas de planejamento Ágil nos painéis, consulte [Visão geral das ferramentas de planejamento Ágil](/help/quicksilver/agile/use-boards-agile-planning-tools/agile-planning-tools-overview.md).

Para obter informações sobre como usar um painel em um projeto, consulte [Gerenciar um projeto na Exibição Ágil](/help/quicksilver/manage-work/projects/manage-projects/manage-projects-in-agile-view.md).

Você também pode acessar quadros no aplicativo móvel. Para obter mais informações, consulte [[!DNL Adobe Workfront] [!UICONTROL Painéis] para dispositivos móveis](/help/quicksilver/workfront-basics/mobile-apps/using-the-workfront-mobile-app/mobile-boards.md).
