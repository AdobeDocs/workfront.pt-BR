# Campos de recurso de assinatura de evento

Os campos de recurso de assinatura de evento representam acionadores para eventos que resultam em uma assinatura de evento enviando uma mensagem de saída para um endpoint configurado. Quando um campo de recurso é editado, um evento UPDATE é acionado.
