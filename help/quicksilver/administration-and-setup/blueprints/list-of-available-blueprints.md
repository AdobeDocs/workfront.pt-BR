# Lista de blueprint disponíveis

<!-- Audited: 5/2025 -->

Os [!DNL Adobe Workfront] blueprints a seguir estão disponíveis para instalação pelos administradores do sistema. Para obter mais informações sobre blueprints, consulte [Visão geral de blueprints](/help/quicksilver/administration-and-setup/blueprints/blueprints-overview.md).

**[!UICONTROL Relatórios de Análise de Usuário Ativos]**
Esse esquema contém um painel para ajudar os administradores a validar a configuração do usuário, identificar desalinhamentos e garantir que as configurações suportem o uso pretendido do aplicativo.&#x200B;

**[!UICONTROL Adicionar Lista de Verificação de TI de Novo Funcionário]**
Este esquema contém um modelo que a organização de serviços profissionais do [!DNL Workfront] usa há anos durante os contratos de implementação com equipes de TI corporativas para organizar atividades de integração de novos funcionários. O uso desse modelo permite que as equipes de TI operem com eficiência, gerando uma nova experiência positiva de funcionário e um rastreamento mais rápido da produtividade.

**[!UICONTROL Planejamento de Orçamento Anual]**
Este esquema contém um modelo que a organização de serviços profissionais do [!DNL Workfront] tem usado por anos durante os contratos de implementação com equipes financeiras para organizar as atividades em torno do planejamento do orçamento anual. O modelo ajuda a definir expectativas com líderes de unidade de negócios e os orienta durante o processo de solicitação e reserva de orçamentos durante o processo de planejamento anual.

**[!UICONTROL Painel da Linha de Base]**
Esse esquema contém um painel para ajudá-lo a tomar decisões de projeto mais rápidas e inteligentes. As linhas de base oferecem a capacidade de comparar o que foi planejado com o que aconteceu, permitindo identificar tarefas com problemas, escopo, deslizamento e tendências de projeto rapidamente. O Painel de Instrumentos da Linha de Base inclui estes relatórios: Linha de Base | Projetos em Andamento, Linha de Base | Projetos concluídos, linha de base | Tarefas em Andamento e Linha de Base | Tarefas Concluídas.

**[!UICONTROL Plano de Entrega de Serviços Contratados]**
Esse esquema contém um modelo que organiza o trabalho relacionado à prestação de serviços voltados para o cliente. Ele foi projetado para criar uma abordagem uniforme para a prestação de serviços contratados.

**[!UICONTROL Plano de Evento Corporativo]**
Este esquema contém um modelo que a organização de serviços profissionais do [!DNL Workfront] tem usado por anos durante os contratos de implementação com equipes de Recursos Humanos para dar suporte às atividades de retenção de funcionários, especificamente eventos corporativos e festas.

**[!UICONTROL Desativar Lista de Verificação de TI de Funcionário Encerrada]**
Este esquema contém um modelo que a organização de serviços profissionais do [!DNL Workfront] usa há anos durante os contratos de implementação com equipes de TI corporativas para gerenciar as várias atividades que devem ocorrer quando um funcionário encerra seu emprego. É essencial desativar o acesso dos funcionários e fazer outras alterações nos sistemas para reduzir a exposição ao risco e garantir que outros membros da equipe possam permanecer produtivos na ausência do funcionário que está partindo.

**[!UICONTROL Produção de Ativo Digital] (Integrada)**
O plano de [!UICONTROL Produção de Ativos Digitais] para processos integrados contém um modelo que organiza o trabalho para um processo de produção digital padrão. O modelo fornece um processo simples para a criação de ativos digitais originais para uma equipe que depende de procedimentos operacionais padrão para definir como realizar o trabalho e usa o projeto apenas para priorizar o trabalho e acompanhar o progresso.

**[!UICONTROL Produção de Ativo Digital] (Gerenciada)**
O pacote de [!UICONTROL Produção de Ativo Digital] contém um modelo que organiza o trabalho para um processo de produção digital padrão. Ela foi criada como um ponto de entrada para gerenciar a criação de ativos digitais no Workfront. Use este pacote para criar uma base para a criação de conteúdo digital e, em seguida, personalize-o para atender às necessidades de outros processos criativos para materiais digitais.

**[!UICONTROL Revisão de Ativo Digital] (Integrada)**
O esquema de [!UICONTROL Revisão de ativo digital] para um processo integrado contém um único modelo para organizar o processo de trabalho associado à revisão de um ativo existente. O modelo fornece um processo de lean manufacturing para uma equipe que depende de procedimentos operacionais padrão para definir como fazer o trabalho e usa o projeto somente para priorizar o trabalho e acompanhar o progresso.

**[!UICONTROL Revisão de Ativo Digital] (Gerenciado)**
O pacote de [!UICONTROL Revisão de ativo digital] contém um único modelo para organizar o trabalho associado a uma revisão de um ativo existente. O modelo fornece um processo simplificado para uma revisão de ativos e, ao mesmo tempo, cria uma experiência consistente em comparação com um novo projeto de produção de ativos.

**[!UICONTROL Editar Lista de Verificação de TI de Funcionário de Transferência]**
Este esquema contém um modelo que a organização de serviços profissionais do [!DNL Workfront] tem usado por anos durante os contratos de implementação com equipes de TI corporativas para gerenciar as várias atividades que devem ocorrer quando um funcionário é transferido para uma nova função. Este modelo reduz os riscos de segurança existentes quando uma transferência de funcionário é concluída sem supervisão de TI. Ele também elimina horas desperdiçadas de tempo improdutivo quando o acesso aos sistemas não é concedido para cumprir as obrigações da nova função do funcionário.

**[!UICONTROL Fechamento Operacional de Fim de Mês]**
Este esquema contém um modelo que a organização de serviços profissionais do [!DNL Workfront] usa há anos durante os contratos de implementação com equipes financeiras para organizar o fechamento dos livros no final do mês. O modelo fornece uma lista de verificação de atividades comuns que devem ser concluídas a cada mês para relatar com precisão e promover a integridade dos negócios.
