## Considerações sobre a atualização para status específicos

* **Ao atualizar um projeto para Concluir:** Certifique-se de que todas as tarefas e problemas estejam concluídos no projeto. Não é possível selecionar o status Concluído de um projeto ou qualquer outro status que seja igual a Concluído quando há tarefas ou problemas que não foram concluídos no projeto. Isso inclui a aprovação de qualquer tarefa ou problema que esteja em um status de Aprovação concluída pendente.
* **Ao atualizar um projeto de Concluído para Atual:** Se todas as tarefas e problemas do projeto forem concluídos, verifique se o Modo de conclusão do projeto está definido como Manual. Se o Modo de conclusão do projeto for Automático, o status do projeto permanecerá Concluído.

## Alterar status do projeto

1. Vá para o projeto cujo status você deseja atualizar.
1. No cabeçalho do projeto, clique no nome do status na **Status** , em seguida, selecione um novo status.

   ![](assets/change-project-status-in-header-drop-down-nwe-350x371.png)

   Ou

   Clique no botão **Mais** menu ![](assets/qs-more-menu.png) ao lado do nome do projeto e clique em **Editar** e selecione um novo status na **Status** e clique em **Salvar**.

   O status do projeto é atualizado para o status selecionado.
