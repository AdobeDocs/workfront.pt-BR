# Alterar o formato de datas em [!DNL Adobe Workfront]

<!--this article used to be called "Change the date format in Adobe Workfront when using Chrome". The team decided to make it more generic and hide the steps. Also see drafted content below-->

<!--Remove me October 2026-->

>[!IMPORTANT]
>
>O procedimento descrito nesta página foi removido porque se aplicou apenas a organizações que ainda não foram integradas ao Adobe Admin Console.
>
>Todas as organizações da Workfront foram integradas à Adobe Admin Console.

Sua preferência de data é controlada pelo seu conjunto de preferências de idioma no Adobe Unified Shell. A configuração de idioma padrão e, portanto, a configuração de data padrão, é `en-US`.

Para obter mais informações, consulte [[!DNL Adobe Unified Experience] para [!DNL Workfront]](/help/quicksilver/workfront-basics/navigate-workfront/workfront-navigation/adobe-unified-experience.md#access-your-profile-and-preferences).
